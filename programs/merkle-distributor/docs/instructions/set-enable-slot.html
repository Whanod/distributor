Set Enable Slot Instruction

The set_enable_slot instruction allows the admin to set the slot number when claiming becomes enabled for a Merkle Distributor.

Overview

This instruction enables the admin to modify the slot number at which users can begin making claims against the distributor. This provides flexibility to delay the start of claiming if needed.

Parameters

  • enable_slot: The new slot number when claiming becomes enabled.

Accounts Required

  1. distributor (write): The Merkle Distributor account to update.
  2. admin (signer): The admin of the distributor who has authority to set the enable slot.

Validation Checks

The instruction performs the following validations:

  1. Verifies the admin is the signer and matches the distributor's admin.

Security Considerations

  • Only the designated admin can set the enable slot.
  • Changing the enable slot affects when users can start claiming tokens.

Notes

  • This instruction is useful if the distribution needs to be delayed after the distributor is created.
  • Users cannot make claims until the current slot is greater than or equal to the enable_slot.
  • The admin can set this to a very high value to effectively pause claiming.
